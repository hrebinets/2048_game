# 2048 Game 🎮  

A simple **2048** game built with **JavaScript, HTML, and CSS**. Slide and merge tiles to reach **2048**!  

## Features  
- 🎯 Swipe to move tiles  
- 🔢 Auto-merging of same-number tiles  
- 🏆 Win by reaching **2048**  
- 💾 Score tracking  

## Tech Stack  
- **JavaScript** – Game logic  
- **HTML + CSS** – UI design  

Play & Enjoy! 🚀
